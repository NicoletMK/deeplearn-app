# DeepLearn 🧠🎬

DeepLearn is a playful and educational web app designed to teach kids and beginners about deepfakes, AI-generated content, and ethical awareness. Users can experiment with creating and detecting deepfakes while reflecting on how technology can be used responsibly.

---

## ✨ Features

### 🧑‍🎨 Creator Mode (GAN Builder)
- Choose a cartoon-style character
- Select a voice line
- Generate a talking-head deepfake video
- Preview results and reflect on how deepfakes are made

### 🕵️‍♀️ Detective Mode *(Coming soon)*
- Spot the deepfake challenges
- Deepfake of the Day
- Multiplayer mode

### 🧭 Ethics Simulator *(Coming soon)*
- Explore “Would it be ethical to…” scenarios
- Learn consequences of using AI unethically

---

## 📸 UI Style

Designed for kids:
- Cartoon-style illustrations
- Big round buttons and playful icons
- Simple step-by-step experience

---

## 🛠️ Tech Stack

- **React + Vite + Tailwind CSS** (Frontend)
- Custom icon-based UI components
- Video & audio previews (HTML5)
- Designed for plug-in integration with tools like [Wav2Lip](https://github.com/Rudrabha/Wav2Lip) or [D-ID](https://studio.d-id.com/)

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/YOUR_USERNAME/deeplearn-app.git
cd deeplearn-app
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the app

```bash
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173)
